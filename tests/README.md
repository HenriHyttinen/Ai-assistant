# Test Files

This directory contains various test HTML files used during development and debugging.

## Files:

- `oauth_test_final.html` - Final OAuth testing page
- `simple_oauth_test.html` - Basic OAuth test
- `debug_oauth.html` - OAuth debugging page
- `final_oauth_test.html` - Final OAuth test
- `google_console_check.html` - Google Console verification
- `oauth_debug.html` - OAuth debugging
- `simple_google_test.html` - Google OAuth test
- `test_oauth.html` - OAuth test page
- `privacy.html` - Privacy policy (for OAuth consent)
- `terms.html` - Terms of service (for OAuth consent)

## Usage:

These files were used during OAuth setup and testing. They can be served locally for testing purposes:

```bash
# Serve test files
python3 -m http.server 8080

# Then access: http://localhost:8080/[filename].html
```

**Note:** These are development test files and should not be used in production.
